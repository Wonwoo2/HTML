1. 배열 slice, splice

- a = ["사과", "포도", "자두", "수박", "복숭아", "오렌지", "파인애플"] - slice(start, end) : start ~ end-1까지의 문자열 추출
- splice(index, n(지울 갯수), a(추가할 요소), c(추가할 요소), d(추가할 요소)) : 수정 - 삽입/추가, 삭제, 삭제후 추가
- splice(3, 0, "메론") : 3번째 위치에 메론을 삽입
- splice(3, 1, "메론") : 3번째 위치의 요소 삭제후 삽입, 여러 개의 요소를 삽입할 수 있음
- a.concat(a1) : a배열과 a1배열을 합침
- a.join() : a배열을 하나의 문자열로 반환
- a2.join() : a2배열을 하나의 문자열로 반환
