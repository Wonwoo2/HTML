의사 클래스(pseudo-class, 가상클래스)
	- 클래스가 정의된 것처럼 간주
	- 특별한 상태나 구조에 따라 선택 
	- :으로 사용
	- 상태 : link, :visited, :hover, :active
	- 구조 : :nth-child(1), :nth-child(2), :nth-child(2n) => 괄호 안의 숫자는 몇번째 자식인가를 나타내고 2n은 2의 배수 번째 자식을 나타냄
 


a:link{
	color : red;
	text-decoration : line-through;		/* underline, overline */ 
}
a:visited{
	color : blue;
	text-decoration : none;
}
a:hover{
	background : yellow;
	font-size : 1.5em;
	text-decoration : none;
}
a:active{
	color : green; 
}
